# Temperature Converter

A simple console application written in C++ that converts temperatures between Celsius and Fahrenheit.

## Concepts Practiced

- Variables
- User input/output
- Conditionals (`if`, `else if`, `else`)
- Arithmetic operations
- Formatting output with `<iomanip>`

## Example

Input:

Temperature: 30
Unit: C

Output:

30.0 degrees Celsius is 86.0 degrees Fahrenheit.
